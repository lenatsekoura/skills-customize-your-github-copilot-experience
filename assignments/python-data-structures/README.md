# 📘 Assignment: Python Data Structures

## 🎯 Objective

Learn how to use Python lists, dictionaries, and loops to store, organize, and process data.

## 📝 Tasks

### 🛠️ List Practice

#### Description
Write functions that work with lists of scores and student data.

#### Requirements
Completed program should:

- Implement `average_score(scores)` to return the average of a list of numbers.
- Implement `top_three_students(students)` to return the names of the top three students by score.
- Use loops and list operations to process the data.

### 🛠️ Dictionary Lookup

#### Description
Write functions that look up student grades and summarize class performance using dictionaries.

#### Requirements
Completed program should:

- Implement `find_student_grade(students, name)` to return a grade or `"Not found"` if the student is missing.
- Implement `class_summary(students)` to return a dictionary with `count`, `min_score`, `max_score`, and `average_score`.
- Handle empty data safely.

### 🛠️ Data Processing with Loops

#### Description
Use loops to filter and transform numeric data.

#### Requirements
Completed program should:

- Implement `filter_passing(scores)` to return all scores that are 60 or higher.
- Implement `scale_scores(scores, factor)` to return a new list with each score multiplied by `factor`.
- Keep the original list unchanged.
