# 📘 Assignment: Games in Python

## 🎯 Objective

Create engaging games in Python to practice programming concepts like loops, conditionals, and user input. You'll build a classic Hangman game where players guess letters to reveal a hidden word, helping you understand game logic and string manipulation in a fun way.

## 📝 Tasks

### 🛠️ Set Up the Game

#### Description
Initialize the game by selecting a random word from a list and setting up variables for tracking guesses and progress.

#### Requirements
Completed program should:
- Include a predefined list of words (e.g., animals, fruits).
- Randomly choose one word at the start.
- Prepare a display for the hidden word (using underscores like _ _ _).
- Set a limit for incorrect guesses (e.g., 6 attempts).

### 🛠️ Handle Guesses and Progress

#### Description
Allow the player to input letter guesses, update the display, and track remaining attempts.

#### Requirements
Completed program should:
- Prompt the player for a single letter input.
- Reveal correct letters in the word display.
- Deduct attempts for incorrect guesses.
- Show the current progress and remaining attempts after each guess.

### 🛠️ End the Game

#### Description
Check for win or lose conditions and display appropriate messages.

#### Requirements
Completed program should:
- End the game when the word is fully guessed (win) or attempts run out (lose).
- Display a congratulatory message for wins (e.g., "You won! The word was [word]").
- Display a game-over message for losses (e.g., "You lost! The word was [word]").
- Optionally, ask if the player wants to play again.
